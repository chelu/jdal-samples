JDAL YUI DataTable Sample Application.
------------------------------------------
http://www.jdal.org

- Create mysql database with scripts in db directory
- Configure database in src/main/resources/jdbc.properties property file
- Compile with maven:  mvn package
- Run with mvn jetty:run
- Connect your brownser to http://localhost:8080/jdal-springmvc-yui-sample


For more information about jdal, see http://www.jdal.org.
